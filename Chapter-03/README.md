# An Introduction to the Graph Panel

## Objectives
This chapter will provide a tour of Time Series Graph panel of the Grafana application. It will give you an overview of the Graph panel, as well as some insight into how to create and connect a data source with Graph panel and customize the Grafana panel settings.
