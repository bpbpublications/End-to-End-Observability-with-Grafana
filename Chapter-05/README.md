# Visualizing Data in the Graph Panel 

## Objectives
This chapter will provide a tour of the visualizing data in the time-series Graph panel of the Grafana application. It will give you information about advanced queries to probe data from Prometheus data source, storing and visualizing time-series data, and leveraging multiple y-axis in comparing multiple units of data.
